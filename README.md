<h2>Actions and Observations</h2>

1. RDP into our "vm-client-1" machine, download and install wireshark, keep only default settings as prompted in installation. Open Wireshark and select the active network interface (Ethernet). In the "Apply a display filter" field, type "icmp" and press Enter. Open PowerShell and Ping the private IP Address of "vm-dc-1". 
```powershell
ping vm-dc-1
```
Observe the traffic in Wireshark. We filtered by ICMP to isolate and observe the echo request and reply packets generated by the ping test.
